# PRD – User Stories

## Authentication

### US-01
As a user, I want to log in using my Google account
so FIN can access my calendar.

**Acceptance Criteria**
- Login via Google OAuth
- WhatsApp ID linked to Google account
- Tokens stored securely

---

## Calendar Management

### US-02
As a user, I want to create calendar events via chat.

Example:
"Create meeting with Andi tomorrow at 10am"

---

### US-03
As a user, I want to see my upcoming schedule.

Example:
"What’s my schedule today?"

---

### US-04
As a user, I want to delete or update calendar events.

---

## Session Handling

### US-05
As a user, I want to stay logged in
without re-authenticating every time.
