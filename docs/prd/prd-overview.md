# PRD – Product Overview

## Product Name
FIN – Your Personal Assistant on WhatsApp

## Vision
Enable users to control Google Workspace through WhatsApp chat,
removing friction from everyday productivity tasks.

> “If you can chat it, FIN can do it.”

## Problem Statement
Users face friction when:
- Switching between WhatsApp and Google apps
- Performing quick actions on mobile
- Using complex automation tools
- Relying on unreliable voice assistants

## Solution
FIN provides:
- Natural language interaction
- Google OAuth-based authentication
- AI-powered intent understanding
- Tool execution via LangChain
