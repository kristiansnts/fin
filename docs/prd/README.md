FIN is a WhatsApp-based AI assistant that allows users to manage Google Workspace
(mainly Google Calendar in MVP) through natural language chat.

This folder contains the official Product Requirements Documents (PRD)
for the MVP phase.

## Stack (MVP)
- WhatsApp Gateway: WAHA
- Backend: Next.js (App Router)
- AI Agent: LangChain JS
- Database: PostgreSQL
- Auth: Google OAuth 2.0

## MVP Focus
- Google Calendar
- WhatsApp chat interface
- Secure authentication
- Session persistence

See each document for detailed specifications.
