# FIN – Technical Requirements Document (MVP)

This folder contains the Technical Requirements Document (TRD) for
FIN – WhatsApp Personal Assistant (MVP).

The TRD defines how the system is built technically and complements
the Product Requirements Document (PRD).

## Scope (MVP)
- WhatsApp (WAHA) integration
- Google OAuth 2.0
- Google Calendar automation
- LangChain JS agent
- PostgreSQL persistence

This document is implementation-ready.
